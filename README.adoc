Zig FFI bridges for cross-language interoperability.

== Bridges

Language-to-language FFI bindings via Zig's C ABI compatibility:

* Ada ↔ Zig
* ATS2 ↔ Zig
* Bebop/V ↔ Zig
* C ↔ Zig
* Gleam ↔ Zig
* Idris2 ↔ Zig
* OCaml ↔ Zig
* Rust ↔ Zig
* Swift ↔ Zig
* Polyglot (multi-language bridge)
