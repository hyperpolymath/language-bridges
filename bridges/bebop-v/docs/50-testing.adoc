= Testing

== Recommended Tests

=== Golden Vectors

Known payload bytes + expected decoded JSON-ish structure.

=== Round-trip

`encode(decode(payload)) == canonical(payload)` (where canonicalization exists)

=== Fuzz

Fuzz decode inputs for crashes (where safe).

=== ABI Layout Checks

Size/alignment checks in Zig/Rust against the header.
