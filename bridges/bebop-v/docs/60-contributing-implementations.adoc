= Contributing Implementations

Implementations live under `implementations/<lang>/`.

The goal: satisfy `include/bebop_v_ffi.h` exactly, using a canonical Bebop implementation.

== Common Pitfalls

* Assuming NUL-terminated strings (use `VBytes`)
* Mismatched struct packing/alignment
* Returning pointers to stack memory
* TCP framing assumptions (must be length-prefixed)
* GUID/UUID byte ordering differences
