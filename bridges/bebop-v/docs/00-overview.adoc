= Overview

`bebop-v-ffi` exposes Bebop serialization to V via a stable C ABI.

The key idea is to *avoid re-implementing the wire format in V*. Instead,
V calls into a small native library (Zig or Rust) that uses the canonical
Bebop tooling/library for correctness.
